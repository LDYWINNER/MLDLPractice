# MLDLPractice

- 머신러닝, 딥러닝 알고리즘 및 데이터 전처리 등 연습을 위한 레포
- 연습할 알고리즘들을 베이스로 한 사이드 프로젝트 준비중

## 구현 연습 목록

* 분류
  - 옷 이미지 다중분류 + 보유 옷(이미지)마다 태그 달기(힙, 댄디, ...)
  - 
  
* NLP
  - 태그: 유사어 찾기 + 달기
  - 

* 추천 알고리즘
  - 옷 추천
    + 날씨, 날짜, 이벤트, 장소, 만나는 사람 등의 정보 기반
    + 태그 데이터 기반
  - 피드
    + 다른 유저들 상태 표시
  - 만족도 분석
    + 추천에 따른 만족도 분석, 학습
    + 옷을 입는 빈도, 비슷한 스타일의 보유한 아이템이 있는지
  - 

* 기타
  - 배경 제거 (외부 api 사용 가능)
